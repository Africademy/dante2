# Initial Content & Read Only

This example cames with a example content. The content must be in the Draft's editor state format. check out the poc.js file in Dante's repo


> Note that this example has a button to toggle the readOnly prop on editor.

---- 
<Playground>
  <Toggle initial={false}>
    {
      ({ on, toggle }) => {
        return <div>
          <Dante 
            content={Lorem}
            read_only={on}
          />

          <button onClick={toggle}>
            read only
          </button>
          {on ? ' si' : ' no' }
        </div>
      }
    }
  </Toggle>
</Playground>

## Content Switch

This example demonstrate how Dante editor react to content prop change

<Playground>
  <Toggle initial={false}>
    {
      ({ on, toggle }) => {
        return <div>
          <Dante 
            content={on ? h1 : Lorem}
          />

          <button onClick={toggle}>
            content switch
          </button>
        </div>
      }
    }
  </Toggle>
</Playground>
